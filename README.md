# Solarized for Beeper

_A Solarized color theme for Beeper._

## Installation

1. Copy `custom.css`
2. Open Beeper
3. Go to `Settings` > `Appearance` > `Open CSS file in editor`
4. Paste the contents of `custom.css` into the editor
5. Save the file
6. Click `Reload CSS`
7. Enjoy!
